--------------**To Run The Application**----------------
Step:1 Download the project
Step2: Install any server(eg:Wamp/Xampp)
Step3: Copy the whole project in the folder www in case of wamp server or in htdocs folder in Xampp server
Step4: Run the server
Step5: Type the url in the browser(eg:localhost/Innoseva)
